# Jimmy's AI 学习手册

[🔗 访问 GitHub Pages](https://jimmy-zhz.github.io)

本仓库基于 [Quartz](https://quartz.jzhao.xyz/) 构建，将 [Obsidian](https://obsidian.md) 笔记发布为静态网站。

- 笔记内容在 [`AI-Knowledge-Base/`](./AI-Knowledge-Base)，用 Obsidian 编辑与管理（该目录即 Obsidian vault 根目录）
- 网站首页内容见 [`AI-Knowledge-Base/index.md`](./AI-Knowledge-Base/index.md)
- 本地预览：`npm i && npx quartz build --serve -d AI-Knowledge-Base`
- 部署：push 到 `quartz` 分支由 GitHub Actions 自动构建并发布到 GitHub Pages
