

## 一、概率基础

[[1.3 概率论与数理统计]]

### 1.1 核心三要素

| 概念 | 定义 | 例子 |
|------|------|------|
| **随机试验** (Random Experiment) | 事前无法确定结果的试验 | 抛硬币、模型预测输出 |
| **样本空间** (Sample Space) Ω | 所有可能结果的集合 | 抛硬币 Ω = {正, 反} |
| **事件** (Event) | 样本空间的子集 | 抛硬币结果为"正" |

### 1.2 概率定义

概率是对随机性（randomness）**和**不确定性（uncertainty）的数学描述。
“机会”（chance）或“可能性”（Likelihood）。

#### 三种定义
```
1. 古典概型：P(A) = 有利结果数 / 所有结果数（等可能性）
2. 频率派：P(A) = lim(n→∞) n_A / n（大量重复试验）
3. 主观概率（贝叶斯派）：基于先验知识的主观置信度
```

#### Sample Spaces 样本空间
一个随机试验的**所有可能结果的集合**。通常用 S 表示
**完整且可穷尽的**

- 抛硬币一次的样本空间为：$S = \{\text{Heads}, \text{Tails} \}$

#### Events of Interest 目标事件
一个**具体的陈述或特定的目标。** ，事实上就是样本空间S的一个**子集**

#### 独立事件 Independent Events 

一个事件的发生，**完全不影响**另一个事件发生的概率。

####  互斥事件 Disjoint events
两个事件**绝不可能同时发生**。

从8个人里抽两个人，满足如下条件：
A：第一个是男人
B：第二个是女人
互斥判定不考虑过程（如对第二次概率的影响），只关注A和B是否有可能同时发生，完全没可能就是互斥。

### 1.3 概率规则 (Probability Rules)

#### 概率的范围规则（Range Rule）

**含义**：任何事件发生的概率绝对不可能小于 0（负数）或大于 1（超过 100%）。0 表示绝不可能发生，1 表示必然发生。

对于任意事件 $A$，$0 \leq P(A) \leq 1$。

#### **Rule 1 — 互斥事件 (Mutually Exclusive Events)**

**概念**：如果两个事件是**互斥的（Disjoint / Mutually Exclusive）**，意味着它们**不能同时发生**（例如：一个人既是 A 型血又是 B 型血是不可能的）。
 $$P(A \cup B) = P(A) + P(B)$$
扩展：$P(A_1 \cup A_2 \cup \cdots) = \sum P(A_i)$

#### Rule 2 — 补集规则 (Complement Rule)
$$\bar{A} = \text{"A 不发生"}, \quad P(\bar{A}) = 1 - P(A)$$用途：求"至少有一个发生"的概率时，用补集更简单

#### **Rule 3 — 加法法则 (Addition Rule)**
概率的总和规则（Sum Rule）：**样本空间中所有可能结果的概率之和必须等于 1**。

**加法公式** ：通用形式（含交集）：
 
$$P(A \cup B) = P(A) + P(B) - P(A \cap B)$$ 
当 A、B 互斥时，交集为 0，退化为 Rule 1
![[Pasted image 20260519093828.png]]
#### Rule 4: Relative Frequency Concept（频率概念）
 大量重复试验中，事件发生的频率趋于稳定值 p：
 $$P(A) \approx \frac{n_A}{N} \quad (N \to \infty)$$
这是概率的**频率派**解释，也是大数定律的现实意义


#### Rule 5 — 独立事件的乘法法则 (Multiplication Rule for Independent Events)

- **核心定义**：两个事件 $A$ 和 $B$ 如果是相互独立（Independent）的，意味着其中一个事件是否发生，**完全不会影响**另一个事件发生的概率。
    
    - _课件例子_：在美国随机挑两个人，第一个人是 A 型血（事件 1）和第二个人是 A 型血（事件 2），这两个人互不认识，所以彼此独立。
        
    - _反例（相关/依赖事件）_：一个拥有 5 个孩子的家庭里随机挑两个孩子，第一个是 A 型血，第二个也是 A 型血。因为是亲兄弟姐妹，基因有相关性，这就**不是**独立事件。



### 1.4 边际概率(Marginal Probability)
$$P(\text{Event}) = \frac{\text{Row or Column Total}}{\text{Grand Total}}$$

**定义：** 指某个特定事件发生的概率，**完全不考虑**其他变量的影响。
区别：边际概率是“看单个事件自己（局部）”，而独立性是“看两个事件之间的关系（交互）”。

- **直观理解**：在统计表中，它通常出现在表格的最边缘（Total 栏），因此得名。
- **课件例子**：
    - 问：随机选一名申请人，他考试分数低 (Low) 的概率是多少？
    - 你不需要管他来自哪种高中，只看“Low”这一行的总计（220人）除以总人数（1000人）。
        $$P(L) = \frac{220}{1000} = 0.22$$
### 1.5 条件概率 Conditional probability
$$P(B|A) = \frac{P(A \cap B)}{P(A)}$$

- 𝑃(𝐴 ∩ 𝐵) ： 联合概率 （交集）
- P（A）： A的边际概率

**定义：** 指在**已知另一个事件已经发生**的前提下，某个事件发生的概率。
![[Pasted image 20260512110842.png]]
- **直观理解**：它缩小了你的“眼界”（样本空间）。你不再看全部 1000 人，而只看满足特定条件的那一拨人。
    
- **符号表示**：$P(A|B)$，读作“在 B 发生的条件下 A 发生的概率”。
- **课件例子**：
    
    - 问：**已知**申请人毕业于优质高中 (Superior)，他考试分数低 (Low) 的概率是多少？
        
    - 这时你的分母不再是 1000，而是优质高中的总人数（500人）。分子是优质高中里分数低的人（55人）。
        
    - $$P(L|S) = \frac{55}{500} = 0.11$$![[Pasted image 20260525160248.png]]

|                                  |**报警器 D 成功报警 (D)**|**报警器 D 失败没响 (not D)**|**总计 (Total)**|
|---|---|---|---|
| **报警器 B 成功报警 ($B$)**             |两个都成功响了|B 响了，但 D 没响|**只有 B 响的总概率**|
| **报警器 B 失败没响 ($\text{not } B$)** |B 没响，但 D 响了|两个都失败（全哑火）|**B 没响的总概率**|
| **总计 (Total)**                   |**只有 D 响的总概率**|**D 没响的总概率**|**1.0 (总概率 100%)**|


- 已知B响了，D也响的概率
$$P(D | B) = \frac{\text{两个都响}}{\text{B响的总数}} = \frac{0.38}{0.95} = \mathbf{0.40}$$

- 已知B没响，D响的概率
	$$P(D | \text{ not } B) = \frac{\text{D响B不响的概率}}{\text{B不响的总数}} = \frac{0.02}{0.05} = \mathbf{0.40}$$

- P(D | B) = P(D | not B) 证明是独立事件
	- B 发生与不发生，D发生的概率相同， 说明B的发生不影响D
	- 不需要反证 D的发生对B的影响，**在数学上，只要证明了“B不影响D”，就自发地、百分之百地证明了“D不影响B“**


### 1.6 独立性（Independence）

- **核心定义**：如果一个事件的发生，**完全不改变**另一个事件发生的概率，那么这两个事件就是相互独立的。
    
- **判别标准**：如果 $P(A|B) = P(A)$，说明条件 $B$ 出现与否，对 $A$ 毫无影响。


### 1.7 贝叶斯定理 ⭐


**贝叶斯定理**（AI工程师面试必考）：

$$P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}$$

**医疗诊断**：
已知“患X病时检测出Y阳性”的概率 $P(阳性|患病)$，去推断“检测出阳性时，真的患病”的概率 $P(患病|阳性)$。

$A = 患病$，$B = 检测结果为阳性$。

| 符号      | 含义              | 例子中的含义                          |
| ------- | --------------- | ------------------------------- |
| P(A)    | 先验概率            | 在做检测之前，此人得X病的概率. （全人群个体对此病的发病率） |
| P(B\|A) | 似然 (Likelihood) | 如果确实得X病，检出Y阳性的概率.               |
| P(A\|B) | 后验概率            | 患者检出Y阳性，他患X病的概率                 |
| P(B)    | 证据 (Evidence)   | 边际概率，全人群中检出Y阳性的总概率 (真阳率和假阳率)    |

## 二、随机变量 (Random Variable)

- **核心定义**：它是一个函数，负责**把随机试验的每一个可能结果，都分配一个唯一的“数字（Numerical Value）”**。

```
随机试验 → 抛硬币      → 结果映射 → X = 1(正), X = 0(反)
           ↑
        这就是随机变量 X
```

- **Parameter（参数）**：指**总体 Population**的特征。它是绝对的、往往是未知的，或者是针对所有对象**统计出来的。
    
- **Statistic（统计量）**：指**样本（Sample）**的特征。它是通过随机抽出来的**一小部分人**计算出来的。

| 类型     | 取值特点       | 描述方式        | 例子              |
| ------ | ---------- | ----------- | --------------- |
| **离散** | 可数、有限或无限可列 | PMF（概率质量函数） | 抛硬币次数、计数数据      |
| **连续** | 无限不可列      | PDF（概率密度函数） | 身高、模型输出的 logits |

**注意**：PDF 的积分 = 1，但 f(x) 可以 > 1（如正态分布在均值处）。

### 2.1 离散型 (Discrete Random Variable)

- **核心定义**：如果一个随机变量的所有可能取值，可以**被一个一个列举出来（A list of distinct values）**，那它就叫做离散随机变量。
- **特点**：取值是**有限个**或**可数的**无限个。就像你在数数。
- **课件例子**：每月收到的客户投诉数量、一天内生产的产品件数、车行卖出的汽车数量。

#### Mean of a Discrete Random Variable（离散随机变量的均值 / 期望值）

它代表了这个随机变量在**长期重复试验下的平均结果（Long-run average value）**。

**期望值（Expected Value，记作 $E(X)$ 或 $\mu_X$）**。
$$\mu_X = \sum x_i p_i = x_1p_1 + x_2p_2 + \dots + x_np_n$$
- $x_i$ : 每一个可能取值
- $p_i$: 每一个x_i发生的概率
- 结果：取**加权平均值**

#### Variance and Standard Deviation（方差与标准差）

- **均值**： 描述“平均水平”或数据中心
- **Median**： 排序后正中间的位置
- **方差和标准差**： 数据离均值有多“散”

**方差（Variance，记作 $\sigma_X^2$）**：偏差平方后的平均
**标准差** ： **方差开平方根**。

####  Binomial Random Variable（二项分布随机变量）

- 描述“非黑即白、只有两种结果”的重复试验。

**必须满足的 4 个硬性条件 
- 一共进行 **n 次独立试验**
- 每次试验只有两种结果：**成功 / 失败**
- 每次成功的概率固定为 **p**
- 各次试验之间相互独立

**既然已经知道了，成功率为P，为什么还要做BPV实验？**

p 描述世界的规则，Binomial 描述规则作用下的不确定结果。

1. 结果的波动
2. 风险/不确定性
	- 会不会很差
	- 最坏是什么
3. **推断 p**
	- 现实中经常 **并不知道 p**， 用观察到的结果X去估计p


在IT行业的应用：

|场景|和 BRV 的关系|本质|
|---|---|---|
|A/B 测试|**最直接（几乎就是 Binomial）**|成功次数建模|
|点击率 CTR 预测|**强相关（建模基础）**|Bernoulli/Binomial → logistic|
|分类模型（classification）|**底层概率假设相关**|Bernoulli likelihood + sigmoid|

### 2.2 连续型(Continuous Random Variable)

- **特点**：取值可以在某个区间内**任意细分**，拥有无限种可能。通常源于测量。
- **课件例子**：咖啡店的日收入（金额可以精确到分甚至更细）、交付包裹所需的时间、公司的股票价格、通勤时间。

#### 正态随机变量（Normal Random Variables）

也就是我们常说的**正态分布（Normal Distribution）**。

在自然界、人类社会以及 AI 训练数据中，**绝大多数连续变量都极其自然地服从或接近正态分布**。

它长什么样？（**钟形曲线 Bell Shape**）
![[Pasted image 20260519110005.png]]
- 对称性（Symmetry）
- 单峰（Unimodal）
- 两头塌、中间鼓（Tapering off）

#### 正态分布的“**黄金三法则**” 
![[Pasted image 20260519110720.png]]
**标准差规则**（**Standard Deviation Rule**，也叫 68-95-99.7 法则）
 1. **68% 法则**（往外走 1 个标准差）
	 **含义**：在这个世界上，有 **68%**（超过三分之二）的成年男性，脚长会落在 9.5 到 12.5 英寸之间。这是最普通、最常见的大多数。
 2. **95% 法则**（往外走 2 个标准差）
	 - **含义**：有 **95%** 的绝大多数人都会落在这个区间内。我们在做数据科学或 A/B 测试时，常常把这个范围以外的数据（也就是那剩下的 5%）看作是“显著不同”或者极端值。
3. **99.7% 法则**（往外走 3 个标准差）
		**含义**：几乎可以说是“天罗地网”了，有 **99.7%** 的人都在这里面。
		也就是说，如果你在温尼伯街头随机找一个人，他的脚长小于 6.5 英寸或者大于 15.5 英寸的概率只有极其微小的 0.3%（千分之三）。
		在工业界（比如著名的 **六西格玛 Six Sigma** 质量管理）中，只要超出了这个 3 倍标准差的范围，产品就会被直接定义为“不合格缺陷品”。

### 2.3 概率分布 Probability Distribution
**The list of possible values along with their associated probabilities is called the probability distribution of a random variable.** 
（所有可能取值的列表，以及它们各自对应的概率，就叫做该随机变量的概率分布。）

#### 概率密度函数（PDF）
[什么是概率密度函数 (PDF)？](https://support.minitab.com/zh-cn/minitab/help-and-how-to/probability-distributions-random-data-and-resampling-analyses/supporting-topics/basics/using-the-probability-density-function-pdf/)

- 它描述了连续随机变量在**某一个具体取值附近**的概率“密度”或集中程度。

- **通俗理解**：它就像是概率分布的“地形图”（山峰高的地方代表数据落在这里的可能性大，低谷代表可能性小）。
#### 累积分布函数 (CDF)
计算 **累积概率**

$$F(x) = P(X \leq x)$$

- CDF 是**单调递增**的
- 无论离散还是连续型都适用
- P(a < X ≤ b) = F(b) − F(a)

## 三、常用概率分布 ⭐⭐⭐

> **AI工程师必须烂熟于心的 6 个分布**。机器学习本质上就是在和这些分布打交道。

### 3.1 离散分布

#### ① 0-1 分布 (Bernoulli)
$$P(X=1) = p, \quad P(X=0) = 1-p$$

**用途**：单次二分类问题、神经网络的二值激活函数。

#### ② 二项分布 Binomial — n次伯努利实验
$$P(X=k) = C_n^k \cdot p^k \cdot (1-p)^{n-k}$$

**用途**：n次独立分类预测中正确次数的分布。

#### ③ 泊松分布 Poisson — 单位时间/空间的稀有事件计数
$$P(X=k) = \frac{\lambda^k e^{-\lambda}}{k\!}$$

**用途**：给定 λ（平均发生率），预测某时段内的发生次数。

> **AI场景**：推荐系统中的用户点击率建模、自然语言处理中的词频建模。

### 3.2 连续分布

#### ④ 均匀分布 Uniform
$$f(x) = \frac{1}{b-a}, \quad x \in [a, b]$$

**用途**：随机初始化权重、蒙特卡洛采样的基础。

#### ⑤ 指数分布 Exponential
$$f(x) = \lambda e^{-\lambda x}, \quad x \geq 0$$

**用途**：描述"等待时间"的分布（如用户下次点击的间隔时间）。

#### ⑥ 正态分布（高斯分布）Normal ⭐⭐⭐
$$f(x) = \frac{1}{\sqrt{2\pi\sigma^2}} \cdot e^{-\frac{(x-\mu)^2}{2\sigma^2}}$$

**用途**：
- 中心极限定理：大量独立随机变量的和趋向正态分布
- 神经网络权重初始化（默认假设）
- 线性回归残差假设
- 高斯核 SVM

**标准正态分布**：μ=0, σ=1，记作 N(0,1)

## 四、数字特征（统计量）⭐⭐⭐

### 4.1 数学期望（均值）E[X]

$$E[X] = \sum_{i} x_i \cdot P(X=x_i) \quad \text{（离散）}$$

$$E[X] = \int_{-\infty}^{+\infty} x \cdot f(x) \, dx \quad \text{（连续）}$$

**性质**：
- E[aX + b] = aE[X] + b
- E[X + Y] = E[X] + E[Y]（线性）
- E[XY] = E[X] · E[Y] 当 X,Y 独立时

### 4.2 方差 Var(X)

$$Var(X) = E[(X - E[X])^2] = E[X^2] - (E[X])^2$$

**性质**：
- Var(aX + b) = a² Var(X)
- Var(X + Y) = Var(X) + Var(Y) + 2Cov(X,Y)
- Var(aX + bY) = a²Var(X) + b²Var(Y) + 2abCov(X,Y)

**标准差**：σ = √Var(X)

### 4.3 协方差与相关系数 ⭐

$$Cov(X,Y) = E[(X - E[X])(Y - E[Y])] = E[XY] - E[X]E[Y]$$

| 指标 | 公式 | 取值范围 | 含义 |
|------|------|----------|------|
| **协方差** | Cov(X,Y) | (−∞, +∞) | 同向变化为正 |
| **相关系数** | ρ = Cov(X,Y) / (σ_X · σ_Y) | [−1, 1] | 标准化后的相关性度量 |

> **相关系数 ρ 的解读**：
> - ρ = 0：不相关（但可能独立）
> - ρ = 1：完全正相关
> - ρ = −1：完全负相关
> - **独立 → 不相关，但反之不成立**

**AI中的应用**：特征选择（剔除高度相关的特征）、数据降维（PCA 求解协方差矩阵）。

## 五、联合分布与边缘分布

### 5.1 联合分布

随机向量 (X, Y) 的联合分布描述两个变量同时取值的概率关系。

- **联合 PMF**：P(X=x, Y=y)
- **联合 PDF**：f(x, y)

### 5.2 边缘分布

从联合分布中"积分掉"一个变量得到另一个变量的分布：

$$f_X(x) = \int_{-\infty}^{+\infty} f(x, y) \, dy \quad \text{（边缘密度）}$$

### 5.3 条件分布

$$P(X|Y) = \frac{P(X, Y)}{P(Y)} \quad \text{（离散）}$$

$$f_{X|Y}(x|y) = \frac{f(x, y)}{f_Y(y)} \quad \text{（连续）}$$

> **贝叶斯网络、隐马尔可夫模型、条件随机场**——这些 ML 模型的核心就是条件分布。

## 六、大数定律与中心极限定理 ⭐⭐⭐

### 6.1 大数定律 (LLN)

当样本量 n → ∞ 时，样本均值收敛到期望：

$$\bar{X}_n = \frac{1}{n}\sum_{i=1}^{n} X_i \xrightarrow{P} E[X]$$

**意义**：训练数据越多，模型学到的均值越接近真实分布均值。**机器学习训练的本质就是大数定律的应用**。

### 6.2 中心极限定理 (CLT) ⭐⭐⭐

无论 X₁, X₂, ..., Xₙ 原本是什么分布（只要独立同分布，方差有限）：

$$\frac{\bar{X}_n - \mu}{\sigma / \sqrt{n}} \xrightarrow{d} N(0, 1)$$

**意义**：
- 样本均值近似服从正态分布 N(μ, σ²/n)
- 这是**假设检验**和**置信区间**的理论基础
- A/B 测试、在线实验的统计显著性分析依赖 CLT

## 九、常用概率分布在 Python（NumPy / SciPy）中的采样

```python
import numpy as np
from scipy import stats

# 伯努利
x = np.random.binomial(1, 0.7, 1000)      # 1次实验，p=0.7，采样1000次

# 二项分布
x = np.random.binomial(10, 0.5, 1000)      # 10次实验，采样1000次

# 泊松
x = np.random.poisson(lam=3, size=1000)   # λ=3

# 正态分布（最重要的！）
x = np.random.normal(loc=0, scale=1, size=1000)  # N(0,1)
x = np.random.randn(1000)                        # 标准正态

# 指数分布
x = np.random.exponential(scale=1/0.5, size=1000)  # λ=0.5
```
