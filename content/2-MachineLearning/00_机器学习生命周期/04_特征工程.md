# 特征工程（Feature Engineering）

> 本手册是机器学习生命周期的关键技术环节，参照 CRISP-DM 框架编写。

---

## 1. 特征工程概述

**特征工程**是机器学习中最重要也最具创造性的环节。其目标是通过选择、创建和转换特征，提升模型的预测性能。

根据 Andrew Ng 的说法："机器学习是一门让计算机从数据中学习的艺术和科学。但在很大程度上，这门艺术在于特征工程。"

特征工程的核心问题是：**"如何从原始数据中创建有意义的特征，使模型能够更好地学习？"

---

## 2. 特征类型

### 2.1 按数据类型分类

| 类型 | 定义 | 示例 | 处理方法 |
|------|------|------|---------|
| **数值特征** | 连续或离散的数值 | 年龄、收入、购买次数 | 缩放、离散化 |
| **分类特征** | 有限类别的值 | 性别、职业、国家 | 编码 |
| **序数特征** | 有顺序的类别 | 学历、满意度、等级 | 有序编码 |
| **时间特征** | 与时间相关的特征 | 年、月、日、星期 | 时间提取 |
| **文本特征** | 自然语言文本 | 评论、标题、描述 | NLP处理 |
| **图像特征** | 图像数据 | 照片、医学影像 | 深度学习 |
| **地理特征** | 地理位置 | 经纬度、地址、城市 | 地理编码 |

### 2.2 按来源分类

| 类型 | 定义 | 示例 |
|------|------|------|
| **原始特征** | 直接从数据集中获取 | 数据表的字段 |
| **衍生特征** | 从原始特征计算得到 | 比率、组合 |
| **外部特征** | 从外部数据源获取 | 第三方数据 |

## 3. 特征创建

### 3.1 数值特征创建

| 方法 | 描述 | 示例 |
|------|------|------|
| **组合特征** | 将多个数值特征组合 | 收入/年龄 |
| **比率特征** | 计算两个特征的比值 | 转化率=点击/曝光 |
| **聚合特征** | 对分组数据计算统计量 | 用户平均消费金额 |
| **多项式特征** | 特征的幂次和交叉项 | $x^2$, $x_1 \times x_2$ |
| **数学变换** | 应用数学函数 | $\log(x)$, $\sqrt{x}$ |

### 3.2 时间特征创建

从时间戳中可以提取丰富的特征：

| 特征类型 | 示例 |
|---------|------|
| **时间单位** | 年、季、月、周、日、时、分、秒 |
| **周期性特征** | 星期几、是否周末、是否节假日 |
| **时间差** | 距离某个事件的时间间隔 |
| **趋势特征** | 过去N天的平均值、增长率 |
| **滚动统计** | 过去7天的累计和、均值 |

### 3.3 分类特征创建

| 方法 | 描述 | 适用场景 |
|------|------|---------|
| **组合类别** | 将多个分类特征组合 | 低基数类别组合 |
| **频率编码** | 用类别频率替换类别 | 捕捉类别稀有性 |
| **目标编码** | 用目标均值替换类别 | 高基数、避免维度爆炸 |
| **类别分箱** | 将稀疏类别合并为"其他" | 高基数类别 |

### 3.4 文本特征创建

| 方法 | 描述 | 适用场景 |
|------|------|---------|
| **词袋模型** | 统计词频或TF-IDF | 短文本、分类 |
| **词嵌入** | 学习词的向量表示 | 语义相似度 |
| **主题模型** | 提取文本主题 | 文档聚类 |
| **命名实体识别** | 提取实体 | 信息抽取 |

### 3.5 交互特征

交互特征捕捉特征之间的相互作用：

| 类型 | 描述 | 示例 |
|------|------|------|
| **乘法交互** | 特征相乘 | $x_1 \times x_2$ |
| **除法交互** | 特征相除 | $x_1 / x_2$ |
| **差值交互** | 特征相减 | $x_1 - x_2$ |
| **条件特征** | 基于条件创建 | 如果$x_1 > 10$则$x_2$否则0 |

## 4. 特征选择

### 4.1 特征选择的目的

- **提高模型性能**：去除噪声特征，提升泛化能力
- **减少过拟合**：降低模型复杂度
- **提高可解释性**：简化模型，易于理解
- **减少计算成本**：降低训练和预测时间

### 4.2 特征选择方法

#### 4.2.1 过滤法（Filter Methods）

基于统计指标选择特征，不依赖模型：

| 方法 | 原理 | 适用场景 |
|------|------|---------|
| **方差阈值** | 删除方差过小的特征 | 去除无信息特征 |
| **相关性** | 删除与目标低相关的特征 | 线性关系 |
| **卡方检验** | 检验特征与目标的独立性 | 分类问题 |
| **互信息** | 衡量特征与目标的信息共享 | 非线性关系 |
| **ANOVA** | 检验特征组间差异 | 分类问题 |

#### 4.2.2 包装法（Wrapper Methods）

使用模型评估特征子集：

| 方法 | 原理 | 优点 | 缺点 |
|------|------|------|------|
| **递归特征消除（RFE）** | 逐步删除最不重要特征 | 准确度高 | 计算量大 |
| **前向选择** | 逐步添加最有价值特征 | 效率较高 | 可能错过最优 |
| **后向消除** | 逐步删除最不重要特征 | 全面 | 计算量大 |

#### 4.2.3 嵌入法（Embedded Methods）

在模型训练中自动选择特征：

| 方法 | 原理 | 适用模型 |
|------|------|---------|
| **Lasso正则化** | L1正则化产生稀疏解 | 线性模型 |
| **树模型重要性** | 基于分裂增益选择特征 | 决策树、随机森林 |
| **特征选择集成** | 使用模型内置的选择器 | 各种模型 |

### 4.3 特征选择流程

```
1. 初步筛选
   ├─ 删除常量特征
   ├─ 删除高相关特征之一
   └─ 删除明显无关特征

2. 特征重要性排序
   ├─ 使用统计指标
   └─ 使用模型重要性

3. 迭代优化
   ├─ 选择top-K特征
   ├─ 训练模型评估
   └─ 调整K值

4. 最终确定
   ├─ 验证选择的特征
   └─ 确认模型性能
```

## 5. 特征提取

### 5.1 降维

当特征维度非常高时，需要降维：

#### 主成分分析（PCA）

- **原理**：找到数据中方差最大的方向，将数据投影到正交空间
- **优点**：无监督、计算高效
- **缺点**：线性变换、可解释性差

#### 线性判别分析（LDA）

- **原理**：找到使类间方差最大、类内方差最小的方向
- **优点**：监督、有类别信息
- **缺点**：只能保留min(C-1, D)维

#### t-SNE/UMAP

- **原理**：非线性降维，保留局部结构
- **优点**：可视化效果好
- **缺点**：不能用于训练、计算慢

### 5.2 特征哈希

- **原理**：将特征映射到固定数量的桶
- **优点**：节省内存、处理高基数
- **缺点**：可能产生哈希冲突

## 6. 特征缩放

### 6.1 为什么要特征缩放

- **加速收敛**：梯度下降在统一尺度下更快收敛
- **避免偏差**：某些算法对特征尺度敏感
- **公平比较**：使不同特征具有可比性

### 6.2 缩放方法

| 方法 | 公式 | 特点 | 适用场景 |
|------|------|------|---------|
| **标准化** | $z = \frac{x - \mu}{\sigma}$ | 均值0，标准差1 | 多数场景 |
| **归一化** | $x' = \frac{x - x_{min}}{x_{max} - x_{min}}$ | 映射到[0,1] | 有明确边界 |
| **Robust Scaler** | 基于IQR的缩放 | 对异常值鲁棒 | 有异常值 |
| **对数变换** | $x' = \log(x + c)$ | 压缩右偏 | 金融数据 |

### 6.3 注意事项

- **训练集上拟合，测试集上应用**：防止数据泄露
- **树模型不需要缩放**：树模型对尺度不敏感
- **距离类算法需要缩放**：KNN、SVM等

## 7. 特征编码

### 7.1 分类特征编码

| 方法 | 描述 | 优点 | 缺点 | 适用场景 |
|------|------|------|------|---------|
| **标签编码** | 用整数替换类别 | 简单 | 引入顺序关系 | 树模型、高基数 |
| **独热编码** | 每个类别创建二元列 | 无顺序 | 维度爆炸 | 低基数、无序 |
| **目标编码** | 用目标均值替换 | 保留信息 | 可能过拟合 | 高基数 |
| **有序编码** | 利用类别顺序 | 保留顺序 | 需要定义顺序 | 序数特征 |
| **频率编码** | 用类别频率替换 | 简单 | 丢失类别信息 | 捕捉稀有性 |
| **嵌入编码** | 学习低维向量 | 丰富语义 | 需要训练 | 深度学习 |

### 7.2 序数特征编码

对于有顺序的类别，需要保留顺序信息：

| 方法 | 描述 | 示例 |
|------|------|------|
| **映射编码** | 手动映射为整数 | 小学=1, 中学=2, 大学=3 |
| **相对编码** | 使用相对大小 | 1, 2, 3表示相对大小 |

## 8. 特征工程最佳实践

### 8.1 特征工程流程

```
1. 理解数据
   ├─ 理解业务背景
   ├─ 理解数据含义
   └─ 理解目标变量

2. 探索性分析
   ├─ 分析特征分布
   ├─ 分析特征与目标关系
   └─ 识别问题特征

3. 特征创建
   ├─ 创建领域特征
   ├─ 创建交互特征
   └─ 创建聚合特征

4. 特征选择
   ├─ 初步筛选
   ├─ 重要性排序
   └─ 迭代优化

5. 特征转换
   ├─ 特征缩放
   ├─ 特征编码
   └─ 特征提取
```

### 8.2 常见陷阱

| 陷阱 | 描述 | 避免方法 |
|------|------|---------|
| **数据泄露** | 使用了未来信息 | 严格按时间顺序处理 |
| **特征冗余** | 高度相关的特征 | 检查相关性 |
| **过度工程** | 创建过多无用特征 | 使用特征选择 |
| **分布偏移** | 训练和测试分布不同 | 验证分布一致性 |
| **缺失值处理不当** | 简单删除或填充 | 根据缺失机制选择方法 |

### 8.3 特征监控

在生产环境中，需要监控特征的质量：

| 监控内容 | 指标 |
|---------|------|
| **数据质量** | 缺失率、异常率 |
| **分布变化** | 与训练集分布的差异 |
| **特征重要性** | 特征贡献度的变化 |
| **模型性能** | 预测准确率的变化 |
