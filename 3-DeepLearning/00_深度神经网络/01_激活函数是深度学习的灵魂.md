

## 深度学习的哲学思考

1. 线性函数的任意组合（无论多复杂）最终仍为线性函数，即 $y=Wx+b$ 的形式；这限制了模型表达复杂非线性关系的能力。
2. 在神经网络中引入**激活函数**，才为模型**注入了灵魂**，使其能够学习复杂的非线性模式。

### 1. 天下大道皆出于y=Wx+b

<font style="background-color: #FFFACD; color:black; padding: 4px; border-radius: 8px; text-shadow: 1px 1px 2px rgba(0,0,0,0.1); font-weight: bold;">大道至简</font>，用哲学的眼光来看， <font style="background-color: #FFFACD; color:black; padding: 4px; border-radius: 8px; text-shadow: 1px 1px 2px rgba(0,0,0,0.1); font-weight: bold;">天下大道皆出于y=Wx+b</font>

*比如 $y_1=Wx+b$, $y_2=W_1y_1+b_1$, 而 $y_1$ 中的 $x$ 可能是多输入，甚至是多项式特征，$W$ 又有可能是矩阵（即 $y_1$ 是多个输出），如此层层嵌套，由简入繁，最终构建出连科学家的大脑也无法想象的复杂表达式。然后通过训练模型学习这个庞大表达式中的权重和偏置，从而产生出强大的预测能力。*

任何物理世界的复杂规律，都是可以由y=wx+b 线性函数+激活函数 非线性化，通过组合变换得到：

<font style="background-color:tomato; color:white; padding:4px; text-shadow: 1px 1px 2px rgba(0,0,0,0.2);font-weight: bold;">万能近似定理（Universal Approximation Theorem）</font>
一个具有至少一个隐藏层的神经网络，只要拥有足够的神经元（即足够的 $y=Wx+b$ 组合）和合适的非线性激活函数，就可以在闭区间内以任意精度拟合**任何连续函数**。

#### 为什么物理世界能被这样拆解？

**微分的思想（泰勒级数）**： 物理学家研究复杂系统时，最常用的武器就是“线性化”。在极小的局部，任何曲线看起来都是直线。神经网络通过层层叠加，本质上是在用海量的“局部线性”去重构“全局非线性”。

**特征的解构**：

物理世界的规律往往是分层的。
- $x$ 层：原子、分子。
- $y_1 = Wx+b$ 层：组合成细胞、组织。
- $y_2 = W_1y_1+b_1$ 层：组合成生物器官、生命活动。
    这种**层级结构**（Hierarchical Structure）恰好对应了“套娃”式线性组合。

**能量与概率的映射**： 物理学中的很多规律（如熵增、量子态）最终表现为一种概率分布。而逻辑回归（线性+Sigmoid）天然就是一种处理概率的工具。

#### 可模拟的边界在哪里？（或者说什么是无法模拟的）

虽然理论上 $y=Wx+b + \text{Activation}$ 可以模拟任何规律，但在实际操作中，它面临两个挑战：

1. **不可计算性**：有些规律（如真随机性或某些量子纠缠态）可能无法通过确定性的算法完美重演。
    
2. **数据依赖（归纳 vs 演绎）**：
    
    - **人类科学家**：通过一个苹果掉落，演绎出万有引力公式（简洁、优美）。
        
    - **AI 模型**：观察了 1 亿个苹果掉落，通过亿万个 $Wx+b$ 模拟出了重力感。它虽然预测得极准，但它并不知道“引力”这个词，它只知道“权重”。


### 2. 激活函数为简单线性回归注入了灵魂
#### 为什么向原函数注入激活函数， 不仅没有失真，反正更向真实世界靠拢了？
原来，我们归纳出来的原函数，本身就不是现实世界真实的数学表达式，而是通过归纳总结和计算得到的近似函数。
![[Pasted image 20260205220019.png]]
例如，我们用线性回归来拟合数据时， 一定会出现噪点和异常。

正是因为==**注入了激活函数，才让模型具备了“无限贴合”现实复杂性的能力。**==

例如：当你喝咖啡时会兴奋，但不喝不一定会不兴奋。喝1杯会兴奋，2杯,3杯会更兴奋，但喝100杯就不会得到100倍的兴奋。现实一定不是线性的。引入恰当的激活函数，如Sigmoid，能将用简单线性函数来总结复杂的非线性关系（比如经过大量训练，模型能察觉当喝到第4，第5杯咖啡时，再喝就不会有更高的兴奋感了， 是激活函数让原本简单的线性函数 有了 拟合真实世界的能力）。
#### 引入激活函数解决了哪些痛点？

1. 非线性 (Non-linearity)：打破“线性套娃”的死循环
	- **痛点**：线性变换的组合（$f(g(x))$，其中 $f$ 和 $g$ 都是线性函数）仍然是线性变换。
	- **必要性**：现实世界的问题（如人脸识别、语言理解）绝不是简单的线性叠加。激活函数给模型注入了"弯曲"的能力。
	- **解决的痛点**：它让模型能够拟合**非线性决策边界**。有了它，模型才能学习复杂的非线性映射关系。
2. 可微性 (Differentiability)：给梯度下降提供“指南针”
	- **痛点**：如果使用阶跃函数（如：小于 0 输出 0，大于 0 输出 1），它在跳变点处不可导，在其他地方导数为 0，无法进行梯度下降。
	- **必要性**：深度学习的核心是**反向传播**，依赖于链式法则。如果激活函数不可微，梯度就无法有效传播。
	- **激活函数的特征**：像 **Sigmoid**、**Tanh** 或 **ReLU** 这类函数，它们在大部分区域都是可微的，提供了明确的梯度方向。
	- **解决的痛点**：使**模型能够通过梯度下降进行训练**。它让误差信号能够沿着网络反向传播，指导权重更新。

#### 为什么当前的AI还不是上帝？
当有了能够拟合复杂模式的"神经元"（线性变换+激活函数），大量神经元通过层级连接形成神经网络，更深的网络结构能够学习更抽象的特征表示。
通过充分的训练和足够的数据，这样的模型能够学习到数据中复杂的统计规律。
然而需要明确的是，当前深度学习模型学习到的是**统计学上的相关性**，而非物理学上的**因果性**。
模型的强大预测能力来自于大规模计算（算力）对数据规律的挖掘，而非对物理本质的理解。

## <font style="background-color:tomato; color:white; padding:4px; text-shadow: 1px 1px 2px rgba(0,0,0,0.2);font-weight: bold;">常见激活函数</font>




### 1. **Sigmoid函数**
![[sigmoid.png]]
	- 🎯：将输入值压缩到==0和1之间==
	- 特点：
		- 所有输出神经元的值非零且和为1，无严格零输出。
		- 确保了最多只有一个神经元具有最高的激活值（==赢家通吃==），而其他的神经元将具有较小的激活值或不激活
	- 应用：常用于二分类问题
	- 公式：$$\sigma(x) = \frac{1}{1 + e^{-x}}$$
	
### 2. **ReLU（Rectified Linear Unit）函数**
![[relu-activation.png]]
	- 定义：输入大于0时输出输入值，否则输出0。
	- [A practical Guide for ReLU](https://medium.com/@danqing/a-practical-guide-to-relu-b83ca804f1f7)
	- 优点：
		-  计算简单，广泛使用
		- 有助于缓解梯度消失问题。
	- 特点：
		- 输入为负时输出严格为零，部分神经元被关闭(不激活)。
	- 公式：$$\text{ReLU}(x) = \max(0, x)$$
	- **ReLU (Rectified Linear Unit)** 是公认的、能**部分缓解梯度消失**的最核心功臣。
### 3. **Tanh函数**
![[tanh-activation.png]]
	- 🎯：将输入值压缩到==-1和1之间==，类似于Sigmoid，但输出范围更大。
	- 应用：常用于循环神经网络（RNN）。
	- 特点：
		- 无论输入正负，输出均在(-1,1)范围内，无非零抑制。
	- 公式：$$\text{tanh}(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}$$

**Tanh 的性能通常优于 Sigmoid**。

- **原因**：Tanh 的输出均值为 0。在神经网络中，如果输入全是正数（如 Sigmoid 的输出），会导致下一层的权重更新出现“锯齿状”摆动，减慢收敛速度。
    
- **结论**：在没有 ReLU 的年代，大家首选 Tanh 而非 Sigmoid。


### 4. **Softmax函数**
![[softmax-fun.png]]
	
- 🎯：将一组输出转换为概率分布，使得每个输出值介于0和1之间，且所有输出值的和为
- 特点：
	- 无论输入正负，输出均在(0,1)范围内，无非零抑制。
	- softmax属于多项逻辑回归分类
- 应用：
	- 常用于多分类问题
	- 在DNN多分类任务的输出层，Softmax 是绝对的王者
- 公式：$$\text{Softmax}(x_i) = \frac{e^{x_i}}{\sum_j e^{x_j}}$$
![[Pasted image 20260216084931.png]]
- 关键知识点（复习重点）
	- **Logits（原始得分）：** 图中的 $z_0, z_1, z_2$ 就是模型为每个类别计算的原始分值（线性方程 $z = wx + b$）。
	- **Argmax（决策算子）：** 它是图中垂直分界线的来源。在任何特征点（横坐标），我们只选取 $y$ 轴数值最大的那条线作为预测结果。
	- **决策边界（Decision Boundary）：** 图中不同颜色圆点的交界处，正是两条直线**相交**的地方（即得分相等，产生竞争）。


### 性能表现的客观排序

- **Loss 下降速度**：ReLU > Tanh > Sigmoid。
    
- **验证集准确率**：ReLU 和 Tanh 显著优于 Sigmoid。
    
- **结论**：在现代深度学习中，除非是输出层需要概率，否则基本**弃用 Sigmoid**。
